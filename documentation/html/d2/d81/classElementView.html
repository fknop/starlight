Description détaillée

Cette classe sert de super-classe à tous les éléments visuels avec lesquels un rayon peut interagir.

Il s'agit plus d'une classe "TAG" pour retrouver le type d'un élément visuel.

Voir également
model/element.h pour son équivalent modèle.
